Estrutura do Site

O HTML 5 é responsável por criar a estrutura do Site, com as tags semânticas para facilitar a leitura do código HTML.

Estilização do Site

O CSS 3 atua na estilização da página, deixando-a mais sofisticada e bonita, criando os estilos em cascata, por meio de seletores.

Dinamicidade do Site

Para gerar dinamicidade a página estática, construída em HTML, usa-se o JavaScript, que é responsável por dar interatividade a página.

Sobre

O curso de JavaScript, foi ministrado, em nove dias, pelo Prof. Heuryk Wylk, no Senac Paulista e que teve uma carga horária total de 36h.

No decorrer das aulas, diversos comandos do JavaScript foram apresentados e implementados no HTML, para gerar dinamicidade as páginas desenvolvidas.

E com base nestas experiências, o projeto, no Curso de JavaScript, tem a finalidade de por em prática as habilidades adquiridas ao longo das aulas, para a conclusão do curso.
